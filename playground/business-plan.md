# DailySeed 사업계획서

## 매일의 씨앗 — 청소년 학습 습관 플랫폼

---

## 1. 요약 (Executive Summary)

DailySeed는 매일 하나의 키워드를 중심으로 뉴스·고전·예술·세계·과학·영어 6개 영역의 교양 콘텐츠를 제공하는 청소년 학습 습관 플랫폼이다.

AI 기반 콘텐츠 자동 생성 파이프라인으로 운영비를 극소화하고, "콘텐츠"가 아닌 "매일 5분 습관"을 핵심 가치로 삼는다.

---

## 2. 문제 정의

### 청소년의 교양 학습 현실
- 입시 중심 교육에서 교양·시사·인문 학습은 후순위로 밀림
- 유튜브·SNS에서 단편적 정보를 소비하지만 체계적 학습으로 이어지지 않음
- 학부모는 자녀의 교양 수준에 관심은 있으나, 직접 커리큘럼을 구성할 여력이 없음

### 기존 솔루션의 한계
- 뉴스레터: 일방적 전달, 읽었는지 확인 불가
- 에듀테크 앱: 수학·영어 등 교과 중심, 교양 영역 부재
- 독서 앱: 긴 호흡 요구, 습관 형성 어려움

---

## 3. 솔루션

### 핵심 컨셉: "하나의 키워드, 여섯 개의 시선, 매일 5분"

매일 하나의 키워드(예: "물")를 중심으로:

| 영역 | 예시 콘텐츠 |
|------|------------|
| 뉴스 | 글로벌 물 부족 위기 |
| 고전 | 노자 "상선약수(上善若水)" |
| 예술 | 모네의 수련 연작 |
| 세계 | 네덜란드의 물 관리 시스템 |
| 과학 | 물 분자의 수소결합 |
| 영어 | water 관련 숙어·어휘 |

### 차별점
1. **연결 학습**: 하나의 주제를 다각도로 보는 구조 (타 서비스에 없음)
2. **극한의 가벼움**: 하루 5분, 부담 제로
3. **부모 연결**: 자녀 학습 현황 리포트 자동 발송
4. **제로에 가까운 운영비**: AI 생성 + 정적 배포

## 4. 시장 분석

### TAM / SAM / SOM

| 구분 | 규모 | 산출 근거 |
|------|------|----------|
| TAM | 2.4조원 | 국내 에듀테크 시장 (2025 기준) |
| SAM | 2,400억원 | 중·고등학생 대상 비교과 학습 시장 (~10%) |
| SOM | 24억원 | 초기 3년 내 점유 가능 규모 (~1%) |

### 타겟 유저
- **Primary**: 중학교 1학년 ~ 고등학교 2학년 (만 13~17세)
- **Decision Maker**: 학부모 (실질 결제자)
- **Secondary**: 학교 교사 (조회 시간, 아침 독서 활용)

### 경쟁 환경 — 솔직한 진단

| 위협 | 심각도 | 대응 |
|------|--------|------|
| AI로 누구나 유사 콘텐츠 생성 가능 | 높음 | 콘텐츠가 아닌 습관·구조로 경쟁 |
| 대형 에듀테크의 교양 영역 진출 | 중간 | 니치 선점 + 학교 채택으로 락인 |
| 무료 콘텐츠(유튜브 등) | 높음 | "정리된 5분" vs "끝없는 스크롤" 차별화 |

## 7. Go-to-Market 전략

### Phase 1: 씨앗 심기 (0~6개월)
- 완전 무료 운영
- 맘카페 (맘스홀릭, 워킹맘) 자연 바이럴
- 교사 커뮤니티 (인디스쿨, 참쌤스쿨) 소개
- 인스타그램 숏폼 콘텐츠 (오늘의 키워드 카드뉴스)
- 목표: MAU 5,000

### Phase 2: 뿌리 내리기 (6~12개월)
- Seed+ 유료 구독 출시
- 부모 리포트를 유료 전환 핵심 훅으로 활용
- 학교 파일럿 5~10곳 확보 (교장/교감 직접 제안)
- 학생 스트릭·뱃지 시스템 도입
- 목표: 유료 전환율 3~5%

### Phase 3: 성장 (12~24개월)
- B2B 학교 영업 확대 (교육청 단위 제안)
- 초등 고학년 버전 출시
- 카카오톡 채널 / 미니앱 연동
- 목표: MAU 50,000

## 13. 결론

DailySeed의 본질적 경쟁력은 콘텐츠가 아니라 **"하나의 키워드 × 여섯 개의 시선 × 매일의 습관"이라는 구조**, 그리고 **거의 0에 가까운 운영비**에 있다.

콘텐츠 레드오션에서 살아남기 위해 "교양 콘텐츠 앱"이 아닌 **"청소년 학습 습관 플랫폼"**으로 포지셔닝하며, 부모 리포트와 학교 채택을 통한 락인 효과로 성장한다.

최악의 경우에도 월 6만원의 운영비만 발생하므로, 실패 리스크가 극히 낮은 사업이다.
